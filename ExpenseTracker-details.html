Expense Tracker iOS Application

An expense tracker app. The main view features a chart that tracks this month’s spending as of date. We can see the history of total spending per specific day. Below, it displays the most recent five transactions that occurred. Each transaction details the merchant, category, date and amount. The amount is negative if it’s a debit or positive and colored if it’s a credit. On the left, we can see the icon associated to the category. These icons are loaded from a third party library SwiftUIFontIcon and mapped through each category. If you click on the See All link, we are redirected to the all transactions screen where they are grouped by month. This app also offers a beautiful dark mode. Beware that this app is data focused.
